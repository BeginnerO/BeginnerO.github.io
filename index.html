HTML Schema

Schema is pretty simple. But you need to link also jquery for using BOSlider.

<html>
	<!--CSS Link-->
	<link rel='stylesheet' type='text/css' href='css/BOSlider.css'>
	<body>
		<!--Your Contents may be here.-->
	</body>
	<!--Jquery Link-->
	<script src='js/jquery.js'></script>
	<!--Javascript Link-->
	<script src='js/BOSlider.js'></script>
</html>

Size

You can easily change height by setting a Height boslider method's parameter. Furthermore you can use both ratio and pixel value. Following examples explain how it works.

HTML code 
<div class='bo-slider'>
		<ul class='bo-content'>
			<li class='bo-each' style='background-image:url("image1.jpg")'></li>
			<li class='bo-each' style='background-image:url("image2.jpg")'></li>
			<li class='bo-each' style='background-image:url("image3.jpg")'></li>
			<li class='bo-each' style='background-image:url("image4.jpg")'></li>
		</ul>
	</div>
Jquery Code 
$('.bo-slider').boslider({
	Height : "3:2"//If you want to send px value, "'value'px".
});

Minimum, Maximum Height

It hard to cover all of device with only ratio setting. Therefore BOSlider provide minimum and maximum height setting. If result value of calculation is smaller(bigger) than min(max) value, height will be min value. Parameters are each Min_Height and Max_Height.

Type of Min_Height and Max_Height is just int, so you don't need to write 'px' unit. If you so, it don't work.

Usage
Example 
$('.other').boslider({
	Min_Height : 250,
	Max_Height : 700,
);

Parameter

Parameter Unit usage default type
Height Pixel,Ratio 150px , 2:1 2:1 String
Min_Height Pixel 160 0 Int
Max_Height Pixel 700 1920 Int

Side Mover

Under Construction

Button that move slide to left, or right. Default value is false, so if you want to add Mover, you need to send parameter Mover.On as true.

Jquery Code 
$('.bo-slider').boslider({
	Mover : {
		On : true,
	},
});

Click Section

You can also modify the clickable section's width. Parameter is Mover.Width.

Jquery Code 
$('.bo-slider').boslider({
	Mover : {
		On : true,
		Width : 35,
	},
});

Gradation

Yes, I know there are people who will hate gradation. So there is option for gradation named Mover.Gradient.

Jquery Code 
$('.bo-slider').boslider({
	Mover : {
		On : true,
		Width : 35,
		Gradient : false,
	},
});

Parameter

Parameter Unit usage default Type
On none true , false false Boolean
Width px 40 20px int
Gradient none true , false true Boolean

Slide Automatically

Setting parameter AutoSlide.On as true will make slide move automatically. And slide interval time can be setted by parameter AutoSlide.Duration .

Jquery Code 
$('.bo-slider').boslider({

		On : true,
	},
	Nav : {
		On : false,	
	},
	AutoSlide { 
		On : true,
		Duration : 5000,
	},
});

Parameter

boButton method need two parameters. First one is CSS rule and second is other properties.

Example 
$.fn.boButton = function(CSS_rule,other_properties){
	CSS_rule : {
		'CSS_properties' : 'value',//no matter how many properties are, they all be applied. 
	},other_properties : {
		Page : [All page], // the page number that above css rules are applied. data type is integer array.
		Which : 'sharp', //the button that above css rules are applied. Either 'sharp' or 'soft'.
		Min_Height : 0, //If height of this slider is smaller than Min_Height value, than css rules will be ignored.
		Max_Height : 10000, //If height of this slider is bigger than Max_Height value, than css rules will be ignored.
	}
}

Name of Property unit of css value css value none String
Other properties
Name Unit usage default Type
Page none [1,3] All page Array
Which none soft or sharp sharp String
Min_Height pixel 300 0 Int
Max_Height pixel 800 10000 Int

Parameter

boTypo method need two parameter. First is CSS Rule and second one is other properties. Belows are more details.

Example 
$.fn.boTypo = function(CSS_rule,other_properties){
	CSS_rule : {
		'CSS_properties' : 'value:[MinWidth|MaxWidth]',//no matter how many properties are, they all be applied. 
		//MinWidth - If width of this slider is smaller than MinWidth value, than css rules will be ignored.
		//MaxWidth - If width of this slider is bigger than MaxWidth value, than css rules will be ignored.
		//If you don't need restriction, just enter value
	},other_properties : {
		Page : [All page], // the page number that above css rules are applied. data type is integer array.
		Which : ['h1','h2','h3','h4','h5','p'], //the tag,class,id that above css rules are applied. It can be everything in div tag
	}
}
