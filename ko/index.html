최대,최소 높이

모든 기기에 맞춰 비율을 설정하는일은 여간 까다로운 일이 아닙니다. 그래서 최대 최소 높이 옵션을 통해 쉽게 경계값들을 제어 하실 수 있습니다. 만약 비율로 계산된 높이 값이 최소(최대) 높이값보다 작다면, 높이는 최소(최대) 높이로 반영됩니다. 매개전달자는 각각 Min_Height 그리고 Max_Height 입니다.

최대, 최소 높이 매개전달자의 자료형은 정수형입니다. 그래서 만약 px이라는 단위를 넣게 될 경우 작동 하지 않습니다.

사용예시
Example 
$('.other').boslider({
	Min_Height : 250,
	Max_Height : 700,
);

Jquery Code 
$('.bo-slider').boslider({
	Mover : {
		On : true,
	},
});

클릭 범위

또한 클릭 가능한 범위를 설정 하실수 있습니다. 초기값은 30이며 매개전달자는 Mover.Width 입니다.

$('.bo-slider').boslider({
	Mover : {
		On : true,
		Width : 35,
	},
});

Parameter

boTypo method need two parameter. First is CSS Rule and second one is other properties. Belows are more details.

Example 
$.fn.boTypo = function(CSS_rule,other_properties){
	CSS_rule : {
		'CSS요소' : '값:[최소Width|최대Width]',//요소들이 몇개든 상관없습니다. 넣고싶은만큼 넣으시면 됩니다. 
		//최소width - 만약 슬라이더의 width가 최소값보다 작다면 css 규칙은 무시 됩니다.
		//최대width - 만약 슬라이더의 width가 최대값보다 작다면 css 규칙은 무시 됩니다.
		//만약 width 제한이 필요없다면 그냥 값만 적어주세요.
	},other_properties : {
		Page : [All page], // 바로 위의 CSS 규칙들을 적용시키고픈 페이지를 정수형 '배열' 형태로 적어주세요. 초기값은 모든페이지입니다.
		Which : ['h1','h2','h3','h4','h5','p'], //the tag,class,id that above css rules are applied. It can be everything in div tag
	}
}
